LiveStreet 0.2

УСТАНОВКА
1. Выполнить SQL дамп(sql.sql), предварительно создав базу данных
2. Настроить коннект к БД(config/config.db.php)
3. Настроить необходимые параметры движка(config/config.php)
4. Дать права 777 каталогам: logs, uploads, templates\compiled, templates\cache
5. Готово! По умолчанию создается учетная запись администратора: admin admin

ОБНОВЛЕНИЕ
Для обновления с версии 0.1.2 необходимо переписать файлы движка поверх старых(обязательно сохраните настройки, а лучше полную копию сайта ДО обновления) и запустить скрипт update/update_0.1.2_to_0.2.php для конвертации структуры БД
